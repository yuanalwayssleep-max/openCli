# 10w炒股专项

这是一个围绕 `10万资金` 搭建的 `AI 辅助 A 股练手项目`。

项目目标：

- 把交易流程固定下来
- 把仓位纪律写清楚
- 把盘前计划和盘后复盘形成闭环
- 用 AI 帮助你整理信息、生成建议、沉淀规则

## 文件结构

- [10万资金A股练手执行计划.csv](/Users/zz/Documents/code/AI/openCli/data/10w炒股专项/10万资金A股练手执行计划.csv)
  用来确定总资金拆分、仓位上限、亏损红线和练手节奏。

- [仓位管理规则表.csv](/Users/zz/Documents/code/AI/openCli/data/10w炒股专项/仓位管理规则表.csv)
  用来规定什么情况下试仓、加仓、减仓、停手。

- [每日交易计划.csv](/Users/zz/Documents/code/AI/openCli/data/10w炒股专项/每日交易计划.csv)
  用来在开盘前填写预期交易计划，包括计划买卖、依据规则、止损止盈和交易后计划。

- [每日复盘记录.csv](/Users/zz/Documents/code/AI/openCli/data/10w炒股专项/每日复盘记录.csv)
  用来在收盘后记录实际交易、执行偏差和下一步打算。

- [练手交易操作说明.md](/Users/zz/Documents/code/AI/openCli/data/10w炒股专项/练手交易操作说明.md)
  用来查看这套项目的完整说明和每日执行流程。

## 每天怎么用

1. 开盘前先看 [10万资金A股练手执行计划.csv](/Users/zz/Documents/code/AI/openCli/data/10w炒股专项/10万资金A股练手执行计划.csv) 和 [仓位管理规则表.csv](/Users/zz/Documents/code/AI/openCli/data/10w炒股专项/仓位管理规则表.csv) 。
2. 再填写 [每日交易计划.csv](/Users/zz/Documents/code/AI/openCli/data/10w炒股专项/每日交易计划.csv) ，明确今天预期做什么、依据什么规则做、止损止盈和交易后怎么处理。
3. 盘中只按计划执行，不随意临盘改规则。
4. 收盘后填写 [每日复盘记录.csv](/Users/zz/Documents/code/AI/openCli/data/10w炒股专项/每日复盘记录.csv) ，记录实际做了什么以及和计划的差异。

## 项目原则

- 第一笔永远是试仓
- 看不懂就轻仓或空仓
- 深套票优先控风险
- 浮盈票优先学会锁利润
- 连续做错就先停手

## 一句话

这个目录不是“选股清单”，而是一套帮助你把 `10万资金练手交易` 做得更有纪律的项目模板。
