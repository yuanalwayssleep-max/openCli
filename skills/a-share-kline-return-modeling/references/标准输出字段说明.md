# 标准输出字段说明

## 1. 每只股票预测结果.csv

| 字段名 | 说明 |
| --- | --- |
| 锚点日期 | 站在这一天收盘后做预测 |
| 训练样本数 | 该锚点日可用于训练的样本条数 |
| 日期 | 该股票对应的锚点交易日 |
| 代码 | 股票代码 |
| 名称 | 股票名称 |
| 行业 | 所属行业 |
| 收盘 | 锚点日收盘价 |
| 预测5日收益率 | 模型预测的未来5个交易日收益率 |
| 预测涨跌 | 根据预测5日收益率正负得到的方向 |
| 实际5日收益率 | 实际未来5个交易日收益率 |
| 实际涨跌 | 根据实际5日收益率正负得到的方向 |
| 涨跌预测是否准确 | 方向预测对为1，否则0 |
| 涨跌幅度差值 | 预测5日收益率与实际5日收益率的绝对差值 |
| 涨跌幅度预测是否准确 | 差值不超过5%记1，否则0 |
| 综合评分 | 当前版本按预测5日收益率换算的排序分数 |
| 未来卖出日期 | 对应5个交易日后的实际卖出日 |

## 2. 每天预测准确率统计.csv

| 字段名 | 说明 |
| --- | --- |
| 锚点日期 | 当天收盘后做预测的日期 |
| 训练样本数 | 当天模型训练样本条数 |
| 股票数量 | 当天输出预测的股票总数 |
| 有实际结果股票数 | 已经可以观察到5日后真实结果的股票数 |
| 涨跌预测准确率 | 全部股票方向预测准确率，百分比格式 |
| 涨跌幅度预测准确率 | 全部股票幅度预测准确率，百分比格式 |
